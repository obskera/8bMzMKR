Local sampled string instruments (offline/local-only)

This folder contains generated WAV sample banks used by Sampler-based instruments:
- violin
- viola
- cello
- bass

Subfolders and required files:

violin/
- G3.wav
- D4.wav
- A4.wav
- E5.wav

viola/
- C3.wav
- G3.wav
- D4.wav
- A4.wav

cello/
- C2.wav
- G2.wav
- D3.wav
- A3.wav

bass/
- E1.wav
- A1.wav
- D2.wav
- G2.wav

These are local sample files. No API calls are used for these instruments.
