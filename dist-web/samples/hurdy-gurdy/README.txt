Local Hurdy Gurdy samples (offline/local-only)

Place CC0 or self-recorded files in this folder with exactly these names:
- C3.wav
- G3.wav
- C4.wav
- G4.wav
- C5.wav

The app uses these files for both live playback and audio export when instrument is "Hurdy Gurdy".
No synthesized oscillator fallback is used for this instrument.
